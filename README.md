### Webpack

Web používá Webpack.
Co to je a jak se to používá najdete zde https://webpack.js.org/

Instalace: v console v rootu spustit
```
npm install
```

Spouští watch
```
npm run watch
```

Generovani produkčne verze JS a CSS
```
npm run build
```
